Offensive Toolkit

12 purpose-built tools for OT/ICS red team exercises spanning 8 industrial protocols.

vulnot-scan

Modbus TCP

Multi-protocol OT network scanner. Discovers live hosts, enumerates Modbus unit IDs, fingerprints services, and detects industrial protocols across target subnets.

Commands

vulnot-scan --target 10.0.0.0/24
vulnot-scan --target 10.0.1.10 --unit-ids 1-10
vulnot-scan --protocol modbus --deep

Key Features

  • Network/subnet enumeration
  • Unit ID discovery
  • Service fingerprinting
  • Protocol detection

vulnot-read

Modbus TCP

Register and coil reader. Dumps holding registers, reads coil states, queries input registers, and generates full register maps for reconnaissance.

Commands

vulnot-read modbus --host 10.0.1.10 --registers 0-100
vulnot-read modbus --host 10.0.1.10 --coils 0-50
vulnot-read modbus --host 10.0.1.10 --map

Key Features

  • Holding register dump
  • Coil status read
  • Input register read
  • Register map generation

vulnot-write

Modbus TCP

Register writer with attack scenarios. Writes arbitrary values to holding registers and coils, and includes pre-built attack scenarios for overflow, setpoint manipulation, logic disruption, and safety bypass.

Commands

vulnot-write modbus --host 10.0.1.10 --reg 100 --val 999
vulnot-write modbus --host 10.0.1.10 --coil 5 --val 1
vulnot-write modbus --scenario overflow
vulnot-write modbus --scenario setpoint-manipulation

Key Features

  • Single/multi register write
  • Coil forcing
  • 4 attack scenarios (overflow, setpoint manipulation, logic disruption, safety bypass)

vulnot-monitor --host 10.0.1.10 --alert --threshold 500

Key Features

  • Historical trending

Key Features
